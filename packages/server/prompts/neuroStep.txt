The project presents an adaptive cognitive-training game for users with existing cognitive challenges, such as memory, coordination, or attention difficulties. The system monitors user performance in real time and dynamically adjusts game complexity to match individual abilities, providing targeted cognitive stimulation. Regular use for approximately 15 to 30 minutes per day is intended to support cognitive improvement and help slow further decline. The solution integrates data-driven analysis with an engaging and accessible user interface to deliver a supportive cognitive-training experience.
